ABOUT CAMM

CAMM or C++ Advanced Mathematics Module is a C++ desktop application developed by two UMASS Lowell students that can help solves a variety of mathematical problems. The program can be downloaded on our Website. Although, we suggested downloading the application as a whole because it will includes all the mathematics functions, each mathematics functions can be individually downloaded separately.
